# 🚀 Guida al Deployment DomusWerk Website\n\n## Deployment su Vercel\n\n### Prerequisiti\n- Account GitHub (hai il repo)\n- Account Vercel (gratuito)\n- Dominio registrato (domuswerk.it)\n\n### Passaggi\n\n1. **Connetti il repo a Vercel:**\n   - Vai su vercel.com\n   - Clicca \"New Project\"\n   - Seleziona `Domuswerk-`\n   - Clicca \"Import\"\n\n2. **Configura variabili d'ambiente in Vercel:**\n   ```\n   NEXT_PUBLIC_DOMAIN=domuswerk.it\n   NEXT_PUBLIC_GOOGLE_ANALYTICS_ID=G-XXXXXXXXXX\n   SENDGRID_API_KEY=SG_XXXXXXXXXX\n   ```\n\n3. **Configura dominio:**\n   - Project Settings → Domains\n   - Aggiungi dominio\n   - Segui istruzioni DNS\n\n4. **Deploy:**\n   - Automatico ogni push a main\n\n---\n\n## Setup Servizi\n\n### Google Analytics\n1. Vai a analytics.google.com\n2. Crea proprietà \"DomusWerk\"\n3. Copia Measurement ID (G-XXXXXXXXXX)\n4. Aggiungi a Vercel env vars\n\n### SendGrid Email\n1. Vai a sendgrid.com\n2. Crea API Key\n3. Copia valore\n4. Aggiungi come SENDGRID_API_KEY in Vercel\n\n---\n\n## Checklist Pre-Launch\n\n- [ ] Repo su GitHub\n- [ ] Deploy Vercel riuscito\n- [ ] Dominio configurato\n- [ ] DNS propagati (24-48h)\n- [ ] Email testate\n- [ ] Modulo contatti funzionante\n- [ ] Mobile responsive\n\n🎉 Done!\n
